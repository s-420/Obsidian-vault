# Interview Guide 项目结构

## 一、项目整体结构

```
interview-guide/
├── app/                              # Spring Boot 后端
├── frontend/                         # React 前端
├── gradle/                           # Gradle 版本目录
├── docker/                            # Docker 配置文件
├── docs/                             # 项目文档
├── docker-compose.yml                # 生产环境 Docker 编排
├── docker-compose.dev.yml            # 开发环境 Docker 编排
└── README.md
```

---

## 二、后端结构 (app/)

```
app/
├── build.gradle                      # Gradle 构建配置
└── src/
    ├── main/
    │   ├── java/interview/guide/
    │   │   ├── App.java             # @SpringBootApplication + @EnableScheduling
    │   │   │
    │   │   ├── common/              # 通用基础能力
    │   │   │   ├── annotation/       # @RateLimit 限流注解
    │   │   │   ├── aspect/           # RateLimitAspect AOP 切面
    │   │   │   ├── async/            # AbstractStreamProducer/Consumer 模板
    │   │   │   ├── ai/               # StructuredOutputInvoker, LlmProviderRegistry
    │   │   │   ├── config/           # 配置类（CORS、S3、ObjectMapper、OpenAPI）
    │   │   │   ├── constant/         # 公共常量
    │   │   │   ├── exception/        # ErrorCode, BusinessException
    │   │   │   ├── model/             # AsyncTaskStatus 等通用模型
    │   │   │   └── result/           # Result<T> 统一响应封装
    │   │   │
    │   │   ├── infrastructure/       # 技术基础设施（可复用）
    │   │   │   ├── export/           # PdfExportService (iText 8)
    │   │   │   ├── file/             # 文件解析、存储、校验、清洗
    │   │   │   ├── mapper/            # MapStruct 映射器
    │   │   │   └── redis/             # RedisService, InterviewSessionCache
    │   │   │
    │   │   └── modules/              # 业务模块（每个模块自包含 MVC）
    │   │       ├── resume/            # 简历管理
    │   │       ├── interview/         # 模拟面试
    │   │       ├── knowledgebase/      # 知识库
    │   │       ├── interviewschedule/ # 面试日程
    │   │       └── voiceinterview/     # 语音面试
    │   │
    │   └── resources/
    │       ├── application.yml        # 主配置文件
    │       ├── voice-interview-opening.yml
    │       ├── logback-spring.xml     # 日志配置
    │       ├── fonts/                 # PDF 中文字体
    │       ├── prompts/               # AI Prompt 模板 (.st)
    │       ├── scripts/               # Redis Lua 脚本
    │       └── skills/                # AI Skill 定义
    │
    └── test/                          # 测试代码
        └── java/interview/guide/
```

## 三、业务模块结构 (以 resume 为例)

```
modules/resume/
├── ResumeController.java            # 控制器（路由）
├── listener/                         # 异步消息
│   ├── AnalyzeStreamProducer.java   # 生产者 → Redis Stream
│   └── AnalyzeStreamConsumer.java   # 消费者
├── model/                           # 数据模型
│   ├── ResumeEntity.java            # JPA 实体
│   ├── ResumeListItemDTO.java       # 列表 DTO
│   ├── ResumeDetailDTO.java         # 详情 DTO
│   └── ResumeAnalysisEntity.java     # 分析结果实体
├── repository/                       # 数据访问
│   ├── ResumeRepository.java
│   └── ResumeAnalysisRepository.java
└── service/                          # 业务服务（按职责拆分）
    ├── ResumeUploadService.java      # 上传
    ├── ResumeParseService.java       # 解析
    ├── ResumeGradingService.java    # 评分
    ├── ResumePersistenceService.java # 持久化
    ├── ResumeHistoryService.java     # 历史
    ├── ResumeDeleteService.java      # 删除
    └── ResumeAnalysisProperties.java # 配置属性
```

## 七、技术栈总结

| 层级 | 技术 | 用途 |
|------|------|------|
| 框架 | Spring Boot 4.0 + Java 21 | 应用框架 |
| AI | Spring AI 2.0 + 阿里云 DashScope | LLM 集成 |
| 数据库 | PostgreSQL + pgvector | 关系 + 向量存储 |
| 缓存/消息 | Redis + Redisson | 会话缓存 + Stream 异步 |
| 文件存储 | S3/RustFS (AWS SDK) | 对象存储 |
| 文档解析 | Apache Tika | PDF/DOCX 解析 |
| PDF导出 | iText 8 | 报告生成 |
| 对象映射 | MapStruct | Entity ↔ DTO |
| API文档 | SpringDoc OpenAPI | Swagger |
| 前端 | React 18 + TypeScript + Vite | UI |
